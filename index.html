Structure

A Specialized Four-Team Framework

1. Workflow Architecture

Leads the design of structured workflows, ensuring all processes are documented.

  • Design, documentation, SOP management
  • Workflow diagrams & data flow models
  • Process optimization

2. Implementation & Automation

Builds and maintains user interfaces & technical integrations.

  • Airtable interfaces & automations
  • Slack, CMS, Drive, Frame.io integrations
  • Custom scripts (Python, JavaScript)

3. Data Reliability & Monitoring

Conducts daily audits and maintains data hygiene.

  • Daily record audits (Airtable CMS)
  • Automated backups
  • Bulk updates & migrations

4. Support Operations

Manages the Service Desk across 7 categories.

  • Bug - Something is broken
  • Access Request - New or change access
  • Clarification - Explain something
  • Data Update - Update data for me
  • Modify Existing - Change what's built
  • New Build - Build something new
  • External Request - From other teams
